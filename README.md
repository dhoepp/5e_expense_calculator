# 5e Expense Calculator

Calculates monthly business income or expenses for a side project in a D&D 5e campaign.

## How to run

```
python3 calc.py
```

## How to use

The calculator will prompt you for:
- Number of side projects
- Three ability score rolls per project
- A d100 roll
- A baseline income value (default 350 gold)

It will then print your monthly profit or expenses in gold, rounded to the nearest 25.
